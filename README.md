# Data Cleaning Portfolio – Python (Pandas)

## Overview
This repository contains multiple data cleaning projects using **Python** and **Pandas**. Each project focuses on transforming raw, messy datasets into clean and analysis-ready formats by applying real-world data preprocessing techniques.

The projects demonstrate skills in:
- Data inspection and exploration  
- Handling missing values  
- Data type conversion  
- Standardization of categorical data  
- Feature engineering (splitting columns)  
- Data validation and consistency checks  
- Outlier detection

### 🎬 Movie Dataset Cleaning Project
**Status:** Completed 
A dataset of the **Top 500 Movies ranked by combined critic and audience scores** is being cleaned.  
Key tasks include:
- Dropping irrelevant columns (e.g., plot, awards, URLs)  
- Handling missing values in critic and audience ratings  
- Creating new features like `score_diff` (difference between critic and audience ratings)  
- Detecting outliers where critic vs audience ratings differ widely  
- Standardizing text fields and fixing categorical inconsistencies using **string operations**  

**Tools Used:** Pandas, NumPy, String library
